# ⚛ Quantum 2D RNG · เครื่องสุ่มตัวเลขควอนตัม 2 หลัก

A beautiful cosmic-themed, bilingual (Thai/English) random 2-digit number generator powered by **TensorFlow.js GPU entropy** and **window.crypto CSPRNG**.

![preview](https://img.shields.io/badge/Entertainment-Only-orange) ![tf.js](https://img.shields.io/badge/TensorFlow.js-4.10-blue) ![lang](https://img.shields.io/badge/Language-Thai%20%2B%20English-green)

---

## ✨ Features

- 🌌 **Cosmic background** — animated stars, nebula clouds, aurora bands, wormhole illusions, shooting stars, mouse parallax
- ⚛ **Quantum entropy** — TF.js GPU `randomUniform` XOR'd with `window.crypto.getRandomValues` (CSPRNG)
- 🕐 **Thailand time sessions** — auto-generates at **11:00 AM** and **2:30 PM** ICT (UTC+7)
- 🔵 **3D countdown rings** — perspective-tilted animated rings for Hours / Minutes / Seconds
- 🌊 **Live quantum wave** — three interference waves driven by crypto entropy every second
- 🇹🇭🇬🇧 **Bilingual** — full Thai + English throughout
- 📱 **Responsive** — works on mobile and desktop

## 🚀 Getting Started

### Option 1 — Open directly
Just open `index.html` in any modern browser. No build step required.

### Option 2 — GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your app will be live at `https://yourusername.github.io/quantum-thai-rng/`

### Option 3 — Local server (recommended)
```bash
npx serve .
# or
python3 -m http.server 8080
```
Then open `http://localhost:8080`

## 🔬 Entropy Method

```
finalNumber = (crypto_val XOR tf_val XOR buf[1]%100 XOR buf[2]%100 XOR buf[3]%37) % 100
```

Where:
- `crypto_val` = `window.crypto.getRandomValues` (OS hardware entropy)
- `tf_val` = `tf.randomUniform([1], 0, 100)` XOR `tf.randomNormal([1], 50, 25)` via WebGL GPU
